通用格式

目录操作命令

  • ls - 列出目录
  • cd - 切换工作目录 (Change Directory)
  • pwd - 查看当前工作目录 (Print Work Directory)
  • mkdir - 创建目录 (Make Directory)

文件操作命令

  • touch - 创建文件
  • cat - 查看文件内容
  • more - 查看文件内容 (可翻页查看, 空格翻页, q退出)
  • cp - 复制文件/文件夹 (copy)
  • rm - 删除文件/文件夹 (remove)
  • mv - 移动文件/文件夹 (move)
