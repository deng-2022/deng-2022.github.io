入门指南：掌握Git的基本操作及进阶学习

本文最后更新于：2023年4月13日 下午

导言

  • Git 是一个分布式版本控制工具，提供代码托管服务

基础

工作区状态

  • 工作区：包含.git 隐藏文件夹的目录，也称为工作目录，用于存放开发的代码
  • 版本库：.git 隐藏文件夹，储存配置信息、日志信息和文件版本信息等
  • 暂存区：.git 隐藏文件夹中的一个 index 文件，也可叫做 stage，用来临时保存修改文件
